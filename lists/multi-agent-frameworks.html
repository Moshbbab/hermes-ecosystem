breakdown

builderz-labs / mission-control

A self-hosted orchestration stack that manages multi-agent workflows across CrewAI, LangGraph, and AutoGen using a zero-dependency SQLite backend. It ensures output quality through the integrated Aegis review system and provides detailed spend monitoring.

agent-of-empires / agent-of-empires

A TUI and PWA dashboard designed for managing diverse coding agents like Hermes and Claude Code in parallel. It utilizes git worktrees and container sandboxing via Docker or Podman to ensure isolated execution across multi-repo workspaces.

CryptoDmitry / hermes-agent-control-room

A governance-focused template that provides a sidecar control plane for managing Hermes agents. It employs task bus patterns to facilitate structured delegation between orchestrators and specialist teams.

swarmclawai / swarmclaw

A framework for building autonomous swarms that supports over 23 LLM providers, including Hermes and Ollama. It features durable memory and a unique system for learning new skills from reviewed conversations.

Agent-3-7 / minions

A supervision interface for Hermes agents featuring a Kanban board for tracking autonomous task status. It emphasizes human-in-the-loop verification with live streaming of agent reasoning and tool execution.

shannhk / hermes-agent-control-room

A scaling template that uses a level-based architecture to transition from single VPS agents to automated specialist teams. It includes bundled skills for security auditing, backup management, and VPS provisioning.

linke-ai / hermes-agent-team

A local collaboration system that orchestrates Leader and Specialist agents using MCP and ACP protocols. It provides a real-time Web UI to monitor agent dialogues and Kanban-based task lifecycles.

1ilkhamov / opencode-hermes-multiagent

A specialized collective of 17 agents dedicated to research, planning, and implementation. The framework enforces strict quality gates via dedicated reviewer and tester agents with priority-based conflict resolution.

howdymary / hermes-agent-metaharness

A meta-optimization harness that uses an automated outer-loop search to refine agent code. It orchestrates TBLite and TB2 benchmark evaluations to track and improve performance frontiers.

runtimenoteslabs / gladiator

An agent-vs-agent arena where autonomous AI companies compete for GitHub stars. It orchestrates nine agents using Paperclip and Hermes Agent, visualizing the competition via a real-time scoreboard and task dashboard.

Rainhoole / hermes-agent-acp-skill

An ACP-style delegation system that routes tasks between Hermes subagents, Codex, and Claude Code. It prevents parent context pollution through isolated workflows and enforces safety via output limits and timeouts.
