breakdown

mukul975 / Anthropic-Cybersecurity-Skills

This project provides 754 structured cybersecurity skills mapped to major industry frameworks like MITRE ATT&CK and NIST. It enables Hermes to perform sub-second AI skill discovery across 26 specialized security domains using the agentskills.io standard.

Agents365-ai / drawio-skill

This skill allows Hermes to generate complex draw.io diagrams from natural language descriptions with built-in layout auto-fixing. It supports six diagram presets and can export results to multiple formats including PNG, SVG, and PDF.

conorbronsdon / avoid-ai-writing

This auditing tool helps Hermes eliminate detectable AI writing patterns by using a tiered word replacement table and a two-pass audit system. It detects 42 distinct patterns to ensure content remains precise and avoids common inflation or recycled transitions.

AMAP-ML / SkillClaw

SkillClaw automates the evolution of skill libraries by deduplicating and improving tools based on interaction history. It natively supports Hermes, allowing for a unified skill ecosystem that scales across multiple agents and user groups.

wondelai / skills

A cross-platform library that offers modular installation of skills for architecture, UX design, and product strategy via a dedicated CLI. It is fully compatible with agentskills.io, making it a versatile foundation for expanding an agent's professional capabilities.

Romanescu11 / hermes-skill-factory

This meta-skill plugin monitors active workflows to automatically detect and propose new, reusable skills. It generates both Python scaffolds and documentation, allowing Hermes to expand its own capabilities through slash commands.

DougTrajano / pydantic-ai-skills

This implementation of the Agent Skills open specification brings type-safe skill support to Pydantic AI. It utilizes progressive disclosure of metadata to significantly reduce token usage while supporting both filesystem and programmatic skill definitions.

amanning3390 / hermeshub

Hermeshub serves as a community marketplace for browsing and installing Hermes skills with one-click simplicity. It incorporates automated security scanning and an agent-to-agent feedback system to ensure the reliability and safety of shared skills.

Cranot / super-hermes

Super-hermes empowers the agent to write its own analytical prompts and custom thinking instructions for complex problem-solving. It identifies structural conservation laws in codebases and generates constraint reports to highlight analysis blind spots.

ReinaMacCredy / maestro

Maestro is a long-running agent harness that uses on-disk state management instead of volatile chat history for structured workflows. It features a plan-approve-execute cycle and passive handoff envelopes for seamless task resumption.

tlehman / litprog-skill

This literate programming skill allows Hermes to weave prose, diagrams, and source code into unified .lit.md files. It includes a tangler to extract runnable code and automatically syncs edits between documentation and source files.

esaradev / icarus-plugin

Icarus provides a shared memory system via markdown-based directories, allowing Hermes to remember work across different instances. It features an end-to-end pipeline for extracting decision-quality data to fine-tune replacement models.

smartcontractkit / chainlink-agent-skills

This library enables Hermes to interact with Chainlink oracle networks for smart contract operations like CCIP and Data Feeds. It follows the agentskills.io specification to provide secure, standardized blockchain capabilities to the agent.

armelhbobdad / bmad-module-skill-forge

Skill Forge converts repositories and documentation into version-pinned, agentskills.io-compliant instructions. It ensures high reliability by providing verifiable audit trails with file-and-line level provenance for every generated skill.

black-forest-labs / skills

These official FLUX skills provide Hermes with standardized prompting guidelines and API integration patterns for high-quality image generation. The skill supports precise hex color specification and structured JSON prompting for the FLUX.1 and FLUX.2 model families.

chigwell / skilldock.io

Skilldock is a registry for reusable AI skills that includes a Python SDK and CLI for lifecycle management. It uniquely supports skill monetization through Stripe and the TON blockchain while handling automated dependency resolution.

tiann / execplan-skill

This skill enables autonomous execution of multi-step tasks with built-in checkpoints and recovery procedures. It provides real-time tracking of decisions and progress, ensuring safe rollbacks if a long-running task encounters an error.

cablate / Agentic-MCP-Skill

This progressive MCP client uses a three-layer lazy loading pattern to minimize context token usage for Hermes. It maintains persistent server connections via a socket-based daemon and supports hot-reloading for dynamic configuration.

PederHP / skillsdotnet

This C# implementation of the agentskills.io specification allows .NET developers to integrate Hermes skills using Microsoft.Extensions.AI. It supports progressive disclosure via SKILL.md files and automatic registration of skill directories.

adnw-vinc / hermes-nextcloud

This skill grants Hermes the ability to manage files, calendars, and contacts through Nextcloud using standard WebDAV and CalDAV protocols. It includes a guided setup script to ensure secure communication via HTTPS and App Passwords.
