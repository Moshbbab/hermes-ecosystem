breakdown

mem0ai / mem0

As the official memory provider for Hermes Agent, Mem0 offers a universal memory layer with multi-level tracking for users, sessions, and autonomous states. It utilizes temporal reasoning and hybrid retrieval to provide time-aware ranking of past and future events across managed or self-hosted environments.

garrytan / gbrain

This opinionated 'brain' for Hermes Agent features a self-wiring knowledge graph that automates typed entity linking through hybrid search. It includes a unique 'doctor' command for autonomous maintenance, ensuring the agent's internal knowledge base remains healthy and remediated.

vectorize-io / hindsight

Hindsight implements a long-term learning framework focused on retain, recall, and reflect workflows for enterprise-grade agent memory. It provides a simple two-line LLM wrapper and supports robust storage backends like PostgreSQL and Oracle AI Database.

plastic-labs / honcho

Honcho is a stateful memory library that uses asynchronous reasoning to extract conclusions and track evolving relationships between users and agents. It serves as the upstream foundation for Hermes-specific wrappers, delivering prompt-ready context via dedicated Python and TypeScript SDKs.

greyhaven-ai / autocontext

This recursive context harness uses a five-role architecture to help agents succeed on complex tasks through iterative self-improvement and structured evaluation. It persists knowledge by producing reusable playbooks, detailed execution traces, and curated datasets.

stephenschoettler / hermes-lcm

A specialized plugin for Hermes Agent that utilizes a hierarchical DAG structure to ensure lossless context management and depth-aware compaction. It provides agents with tools to search and expand compressed history stored in a persistent SQLite backend.

AxDSan / mnemosyne

A zero-dependency memory system designed for sub-millisecond latency in local Hermes Agent environments. It uses an SQLite-backed MEMORIA engine to manage structured fact triples and temporal reasoning with native vector search.

elkimek / honcho-self-hosted

This project provides a self-hosted backend for Honcho, enabling private cross-session memory persistence specifically for Hermes Agent users. It supports local inference servers and includes automated Docker scripts for easy deployment of PostgreSQL and Redis components.

yoloshii / ClawMem

ClawMem is a local-first context engine that operates entirely on-device without cloud dependencies or API keys. It integrates via MCP and agent plugins, utilizing a combination of BM25, vector search, and knowledge graphs for hybrid retrieval.

plur-ai / plur

Plur is a shared memory layer that uses an open YAML 'engram' format to facilitate zero-cost local learning for multi-agent systems. It implements ACT-R activation principles to strengthen relevant facts and decay stale information over time.

EfficientContext / ContextPilot

Focused on inference optimization, ContextPilot reduces input tokens by reordering and deduplicating shared context blocks for Hermes and OpenClaw. It significantly boosts prefill throughput, achieving up to 3x faster processing speeds while preserving accuracy.

keepnotes-ai / keep

Keep provides reflective memory by combining semantic search with ranked graph traversal and automated indexing of local files and URLs. It uses edge tags to create navigable relationships with automatic inverse links for structured context discovery.

yepyhun / Brainstack

Brainstack consolidates temporal continuity and graph truth into a five-shelf architecture to prevent the 'flat memory blob' issue. It functions as a native Hermes-Agent MemoryProvider plugin powered by SQLite and the Kuzu graph database.

amanning3390 / flowstate-qmd

This anticipatory memory system prefetches context by monitoring agent session logs to predict future information needs. Built for the Hermes 2026 Hackathon, it features local-first storage using SQLite-vec and an MCP-native server for coding clients.

alias8818 / hermes-tool-slimmer

This utility reduces tool-schema token overhead for Hermes Agents by using BM25 ranking and keyword selection. It includes a diagnostic dashboard to track real-time token savings across native tools and MCP integrations.
