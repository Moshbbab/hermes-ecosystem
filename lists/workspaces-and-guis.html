breakdown

farion1231 / cc-switch

A cross-platform manager built with Tauri 2 that provides a unified interface for Hermes Agent, Claude Code, Gemini CLI, and other agent CLIs. It simplifies the management of multiple AI tools across Windows, macOS, and Linux.

iOfficeAI / AionUi

A local cowork application that integrates Hermes Agent and 13+ other CLI agents into a single GUI. It features a built-in engine for web search and MCP tool integration, with the ability to automatically generate editable Word, Excel, and Morph PPT documents.

nesquena / hermes-webui

A lightweight web-based interface built with Python and vanilla JavaScript that maintains full feature parity with the Hermes CLI. It utilizes a three-panel layout for efficient workspace file browsing and inline previews.

fathah / hermes-desktop

A desktop companion app designed for the installation and configuration of the Hermes Agent backend. It includes support for 16 messaging gateways, multiple AI providers, and integrated tools for persona editing and scheduled tasks.

EKKOLearnAI / hermes-web-ui

A comprehensive web dashboard focused on session management and multi-platform configuration for 8 messaging services like Discord and Slack. It provides detailed usage analytics for token tracking, cost estimation, and model distribution.

outsourc-e / hermes-workspace

A native web workspace featuring a PTY terminal and Monaco editor for full file system access. It introduces Swarm Mode for role-based dispatch using tmux-backed workers alongside an integrated MCP catalog and memory browser.

dodo-reach / hermes-desktop

A native macOS workspace that connects directly via SSH without the need for browser wrappers or daemons. It provides dedicated views for Kanban boards, cron jobs, and session data across multiple Hermes profiles.

xaspx / hermes-control-interface

A self-hosted web dashboard featuring the ZOO Swarm Monitor with a kanban pipeline and live event feed. It offers a full MCP server control plane with log tailing and a browser-based terminal for system management.

jazzyalex / agent-sessions

A native macOS session browser and 'agent cockpit' for monitoring live sessions and usage limits. It provides local-only indexing for searching AI agent histories without any remote data uploads or telemetry.

nickvasilescu / hermes-desktop-os1

A macOS interface specifically designed for Hermes Agent instances running on Orgo cloud computers and SSH hosts. It features one-click automated installation on cloud VMs and integrated WebRTC voice mode via OpenAI Realtime.

Eynzof / hermes-agent-cn-desktop

A Windows-first desktop application built with Tauri, Rust, and TypeScript, tailored for the Hermes-CN-Core runtime. It includes health checks, update management, and integration with Feishu.

liftaris / herm

An alternative TUI and dashboard built with OpenTUI that renders markdown, LaTeX, and tool calls. It uses Kanban boards to monitor agentic work and provides terminal-based management of profiles and environment configs.

clawvader-tech / hermes-telegram-miniapp

A mobile-first React SPA designed as a Telegram Mini App with a terminal-style UI for agent management. It implements secure HMAC-SHA256 and Ed25519 authentication and supports vision analysis for file attachments.

aivrar / portable-hermes-agent

A zero-install Windows desktop GUI that runs without Docker or admin rights, integrating local models via LM Studio. It comes pre-loaded with over 100 tools, including ComfyUI and a dynamic tool maker for creating API wrappers.

pyrate-llama / hermes-ui

A glassmorphic web interface consisting of a single-file React 18 application and a lightweight Python proxy. It provides real-time log streaming and a live workspace file browser for self-hosted assistants.

sanchomuzax / hermes-webui

A process monitoring and configuration dashboard that tracks costs and model distribution via a real-time activity feed. It allows for read-only inspection of skill sources and full-text search of conversation histories.

Euraika-Labs / pan-ui

A self-hosted workspace that combines a chat-first interface with tool approval cards and SSE streaming. It includes an integrated marketplace for discovering MCP servers and isolated profile management for separate API keys and memory.

Felix-Forever / hermes-agent-desktop

A multi-agent desktop client that orchestrates collaboration between 20 specialized AI agents via a PM orchestrator. It features a visual Skill Store with over 50 curated one-click installations and streaming SSE responses.
