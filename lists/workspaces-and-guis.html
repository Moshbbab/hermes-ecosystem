breakdown

farion1231 / cc-switch

Built with the Tauri 2 framework, this cross-platform manager provides a unified desktop entry point for Hermes Agent alongside other major CLI tools like Claude Code and Gemini CLI. It simplifies the lifecycle of multiple AI agents across Windows, macOS, and Linux within a single application.

iOfficeAI / AionUi

AionUi is a local cowork application that provides a unified interface for multiple agent CLIs with direct file system access for coding tasks. It distinguishes itself by offering automated generation of editable office documents, including PowerPoint, Word, and Excel files.

nesquena / hermes-webui

This zero-build web interface achieves full feature parity with the Hermes CLI using only Python and vanilla JavaScript. It features a functional three-panel layout that combines workspace file browsing with inline previews for a seamless browser-based experience.

fathah / hermes-desktop

This desktop companion simplifies the setup process with guided installations for both local and remote backends. It offers a comprehensive GUI for managing 16 different messaging gateways, scheduled tasks, and custom agent personas.

EKKOLearnAI / hermes-web-ui

This dashboard focuses on multi-platform connectivity, allowing users to configure Hermes for Telegram, Discord, Slack, and WhatsApp from a single location. It includes real-time chat streaming with tool call inspection and detailed analytics to track token consumption and costs.

outsourc-e / hermes-workspace

Designed for complex workflows, this workspace features a 'Swarm Mode' for orchestrating multiple autonomous, role-based agents simultaneously. It provides a native web environment that integrates a terminal, memory browser, and skill manager without requiring forks of the core Hermes Agent.

dodo-reach / hermes-desktop

This native macOS workspace connects directly to Hermes hosts via SSH to manage remote sessions and cron jobs. It includes built-in Kanban boards for project management and supports multi-agent workflows through selectable Hermes profiles.

xaspx / hermes-control-interface

This self-hosted dashboard provides deep system integration with systemd controls and real-time logs for managing agent gateways. It stands out with its detailed token analytics that track performance and costs across different models and tools.

jazzyalex / agent-sessions

A privacy-focused macOS session browser that indexes agent histories locally without telemetry or remote logging. It includes an 'Agent Cockpit' specifically designed to monitor active iTerm2 sessions and track usage limits across various coding agents.

nickvasilescu / hermes-desktop-os1

This macOS workspace is tailored for Orgo cloud computers, offering one-click automated installations of Hermes Agent on fresh cloud VMs. It integrates WebRTC voice mode and Orgo MCP to enable real-time vocal interactions with the agent.

clawvader-tech / hermes-telegram-miniapp

This mobile-first React SPA brings a terminal-style dashboard to Telegram as a Mini App. It features secure HMAC-SHA256 authentication and supports advanced file interactions, including automated vision analysis and OCR capabilities.

Eynzof / hermes-agent-cn-desktop

Built with Tauri and Rust, this desktop application provides a native Windows and macOS experience optimized for the Chinese user base. It includes integrated management for the Hermes kernel, MCP tools, and agent memory within a localized interface.

aivrar / portable-hermes-agent

This zero-install Windows GUI requires no admin rights and comes pre-loaded with over 100 tools, including ComfyUI and MusicGen. It features a dynamic tool maker that allows users to create their own API wrappers and Python handlers without complex configuration.

pyrate-llama / hermes-ui

This lightweight, glassmorphic web interface uses a single-file React application and a Python proxy for easy self-hosting. It provides real-time log streaming and a live workspace file browser to monitor agent activity and manage skills.

sanchomuzax / hermes-webui

This dashboard focuses on process monitoring and historical analysis through a real-time activity feed and full-text conversation search. It provides a read-only inspection environment for auditing configuration files and agent skill sources.

Euraika-Labs / pan-ui

Pan-ui is a chat-first workspace that utilizes SSE streaming and tool approval cards for interactive agent control. It features an integrated marketplace for discovering skills and MCP servers, along with isolated profile management for separate API keys and sessions.

Felix-Forever / hermes-agent-desktop

This desktop client is built for orchestration, managing collaboration between up to 20 specialized AI agents. It includes a visual 'Skill Store' for one-click installations of over 50 curated capabilities and supports real-time model switching.
